# Privacy Interface

Privacy behavior applies to agents, extensions, tools, drivers, exports, and generated stores.

The system must ask confirmation before reading secrets, using external accounts, writing sensitive durable memory, making private material visible outside the intended subject boundary, deleting, redacting, tombstoning, or writing to kernel space.

Released, redacted, tombstoned, and deleted material must be respected by runtime stores and caches.

AI Workroot uses permission hints, not a heavy permission system.
