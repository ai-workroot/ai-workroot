# Task Lifecycle

## Start

Create or locate a task.

## Work

Keep `brief.md`, `decisions.md`, `index.md`, and `todo.md` current.

## Close

Save outputs, update indexes, and promote what matters into Mind.

Before a task is considered closed:

- `brief.md` should contain the final effective summary
- `handoff.md` should be short enough for a future agent to resume or confirm closure
- `decisions.md` should contain important choices
- `index.md` should link outputs, evidence, and promoted Mind entries
- noisy process material should move to `archive/`
- reusable lessons should move to `space/mind/`

Future agents should not need to read old scratch history by default.
