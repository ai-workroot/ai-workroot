# Profile

`space/profile/` defines who this Workroot represents.

This is the first required setup step.

AI Workroot needs a subject before formal work begins. Without a subject, tasks, memories, knowledge, values, and preferences have no stable center.

The subject can be:

- a person
- a team
- a role
- a project
- an organization

Start with:

- `profile.md`
- `roles.md`
- `values.md`
- `preferences.md`

## Minimum Identity

At first, answer only what is needed:

1. Who or what does this Workroot represent?
2. What role should the AI play?
3. What direction, work, or life area should it support?
4. What values, boundaries, or preferences should it respect?
5. What should the AI avoid misunderstanding?

The identity can be broad or specialized. It is not a prison; it is the first anchor. It should evolve as work, memory, knowledge, and reflection accumulate.

Keep this layer concise. Detailed memories, knowledge, decisions, and reflections belong in `space/mind/`.
