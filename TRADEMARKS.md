# Trademark Policy

AI Workroot is open-source software, but the project name and brand must stay clear and trustworthy.

This policy is intended to let people use, fork, teach, discuss, and build with AI Workroot while preventing confusion about what is official.

## Protected Names

The following names and identifiers are reserved by the AI Workroot project:

- AI Workroot
- Workroot, when used to identify this project or a compatible product
- aiworkroot.com
- future AI Workroot logos, marks, icons, slogans, and visual identity assets

The Apache License, Version 2.0 does not grant trademark rights.

## Allowed Uses

You may use the names in truthful, non-confusing ways, such as:

- "built with AI Workroot"
- "based on AI Workroot"
- "compatible with AI Workroot"
- "a fork of AI Workroot"
- "uses the AI Workroot protocol"
- academic, educational, journalistic, or community discussion

These uses must not imply endorsement, sponsorship, official status, or ownership by the AI Workroot project.

## Uses That Need Permission

You need separate permission to:

- use "AI Workroot" as the name of a product, company, hosted service, app, plugin, marketplace, or commercial offering
- use names such as "AI Workroot Cloud", "AI Workroot Pro", "Official AI Workroot", or similar names that imply official status
- use the project domain or future logos in a way that suggests endorsement
- register domains, social accounts, packages, trademarks, or service marks that are confusingly similar to AI Workroot
- distribute a modified version while presenting it as the official AI Workroot release

## Forks And Modified Versions

Forks are welcome.

If you distribute a modified version, use a distinct name unless you have explicit permission to use the AI Workroot brand.

It is fine to say that your work is based on AI Workroot, but the main product or service name should make clear that it is your own project.

## Commercial Use

The Apache-2.0 license allows commercial use of the code and documentation.

Commercial use of the code does not automatically grant the right to use the AI Workroot brand for a commercial product or service.

This separation is intentional:

- the core should remain open and useful to everyone
- the official project identity should remain clear
- future official commercial offerings should not be confused with third-party products

## Brand Assets

AI Workroot does not currently include a final logo or visual identity asset.

Future logos, icons, screenshots, design systems, or brand assets may have additional usage rules.

## No Confusion Rule

When in doubt, ask:

> Would an ordinary user reasonably believe this is official AI Workroot?

If yes, use a different name or ask for permission.
