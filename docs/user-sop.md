# User SOP

This is the simple operating manual for ordinary users.

You do not need to understand the project structure before using AI Workroot.

Your job is simple:

```text
say what you want to do -> work with AI -> save what matters -> continue next time
```

## Step 1: Put It Somewhere

Download or clone the project, then put the folder somewhere you can keep using it.

Examples:

```text
Documents/AI/my-study-helper
```

```text
Work/my-workspace
```

```text
Projects/team-release-helper
```

You may rename only the outer folder. Choose a name that feels natural.

## Step 2: Open It With AI

Open the folder with Codex, Claude Code, or another capable AI agent.

Say:

```text
I want this workspace to help me with [area]. Please set it up with me, then help me start my first real task.
```

If you already have a real problem, say that instead:

```text
I have meeting notes and many follow-ups. Help me organize them.
```

```text
I am in grade 7. Help me learn fractions and give me practice questions.
```

The AI should ask at most one or two simple setup questions, then help you work.

The AI should answer in the language you use. If you ask it to use another language, it should use that language.

## Step 3: Understand Tasks Simply

A task is just the thing you are working on now.

Examples:

- learn one topic
- finish one homework plan
- organize one meeting
- write one chapter
- check one release
- plan one month of business work

You can still ask quick questions. If a quick question grows into something bigger, the AI should help turn it into a task so you can continue later.

## Step 4: Work Normally

Say what you want in normal language.

For a quick question:

```text
This is a quick question. Please answer directly.
```

For a task:

```text
This is a task I want to continue later. Help me keep track of it.
```

For learning:

```text
Help me understand this topic, then save what I should practice next.
```

For a decision:

```text
Help me compare the options and decide what to do.
```

## Step 5: End A Task Well

When the task is not finished:

```text
Leave a clear next step so we can continue next time.
```

When the task is finished:

```text
This task is finished. Save what matters and help me start a new task.
```

When something is useful:

```text
Save what matters.
```

The AI should save useful results, decisions, lessons, and next steps without making you choose where they go.

## Step 6: Continue Later

Next time, open the same folder and say:

```text
Help me continue.
```

The AI should remind you:

- what task you were working on
- what was finished
- what matters now
- what to do next
- what sentence you can say next

## Step 7: Review Past Work

You can ask:

```text
What tasks have I done before?
```

```text
Summarize what we finished recently.
```

```text
What useful notes have we saved?
```

The AI should use the local workspace records to list or summarize your past tasks. If something was not saved, it should say that clearly.

## Good First Things To Try

Personal:

```text
Help me plan this week and choose the most important task.
```

Student:

```text
I am in grade 7. Help me understand this lesson and make practice questions.
```

Office work:

```text
Help me turn these meeting notes into action items and an email draft.
```

Team:

```text
Help us check this release and make a simple risk list.
```

Writing:

```text
I want to write a story, but I only have scattered ideas. Help me start.
```

Founder:

```text
I run a small company and do not know where to start. Help me make a 30-day plan.
```

## What You Do Not Need To Learn First

You do not need to manually:

- create folders
- choose categories
- edit handoff files
- decide where knowledge is stored
- manage databases or caches

You only need to work, review the result, and tell the AI what should be kept.
