# Promotion Rules

Promote only what has future value.

- meaningful experience -> `space/mind/memory/`
- reusable understanding -> `space/mind/knowledge/`
- operating commitment -> `space/mind/principles/`
- important choice -> `space/mind/decisions/`
- repeated signal -> `space/mind/patterns/`
- review -> `space/mind/reflections/`
- obsolete belief -> `space/mind/invalidated/`
- released past context -> `space/mind/released/`

When releasing painful context, preserve the lesson first when useful.

Release should make unnecessary pain quiet, not erase useful responsibility, decisions, or knowledge.

When the user wants remembrance without reactivation, use `tombstone`: keep a minimal marker and preserve the lesson elsewhere.
