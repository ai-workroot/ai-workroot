# Memory Model

AI Workroot uses layered memory:

- memory: what happened
- knowledge: reusable understanding
- principles: durable rules and values
- decisions: important choices
- patterns: repeated signals
- reflections: reviews and deeper thinking
- invalidated: what should no longer be believed
- released: what the subject chooses not to actively carry forward
- tombstone: a released marker for remembrance without normal retrieval

The model is a network, not a flat folder tree.
